# twitter-slack-adapter

## Usage

```
$ heroku config:set HUBOT_TWITTER_KEY=***
$ heroku config:set HUBOT_TWITTER_SECRET=***
$ heroku config:set HUBOT_TWITTER_TOKEN=***
$ heroku config:set HUBOT_TWITTER_TOKEN_SECRET=***
$ heroku config:set SLACK_TOKEN=***
$ heroku config:set SLACK_CHANNEL=***
$ git push heroku master
```
